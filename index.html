PROJECTS I'VE DONE

Travel Booking System

It is developed to be used by bus travel agencies and the customers. Travel agencies use this project for maintaining their bus trip schedules for daily buses or for some days of the week between different cities they serve. The operators, as well as the customers, can reserve tickets through this system. Various filters and advance booking facilities are also included in the website. It also shows the details about bus types. Trip status displays the seats in the graphical view of seating arrangement whether any seat is available or not.

Technologies: Java Servlet, struts, MVC, MySQL

ToneLiber

It is an online music streaming website, it uses AJAX to send and receive the data, thus, it is very fast and no refreshing of a website is needed during the user session except for signing in. Different artists can upload their individual soundtracks or complete albums (with the options like genre, album art, tags, etc.). The website user can create & update their profile. Users can listen, like, comment, share and download the individual track or album. They can also create their own playlist from the available tracks and share them on various social media websites.

Technologies: Java Servlet, struts, MVC, Ajax, MySQL
